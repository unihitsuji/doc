Minor modes enabled in this buffer: Auto-Save Font-Lock このバッファで有効なマイナーモード: Auto-Save Font-Lock

The major mode is Scheme mode defined in scheme.el: メジャーモードは scheme.el で定義された Scheme モードです:

Major mode for editing Scheme code. Scheme コードを編集するためのメジャーモードです。 Editing commands are similar to those of lisp-mode. 編集コマンドは lisp-mode と似ています。 In addition, if an inferior Scheme process is running, some additional commands will be defined, for evaluating expressions and controlling the interpreter, and the state of the process will be displayed in the mode line of all Scheme buffers. さらに、下位の Scheme プロセスが実行されている場合、 式の評価やインタープリタの制御のための追加コマンドが定義され、 プロセスの状態がすべての Scheme バッファのモードラインに 表示されます。 The names of commands that interact with the Scheme process start with "xscheme-" if you use the MIT Scheme-specific xscheme package; for more information see the documentation for xscheme-interaction-mode. MIT Scheme 固有の xscheme パッケージを使用する場合、 Scheme プロセスと対話するコマンドの名前は "xscheme-" で始まります。 詳細については xscheme-interaction-mode のドキュメントを 参照してください。 Use M-x run-scheme to start an inferior Scheme using the more general cmuscheme package. より一般的な cmuscheme パッケージを使用して 下位の Scheme を起動するには M-x run-scheme を使用します。

Commands: Delete converts tabs to spaces as it moves back. Blank lines separate paragraphs. Semicolons start comments. コマンド: Delete は戻るときにタブをスペースに変換します。 空行は段落を区切ります。セミコロンはコメントを開始します。
