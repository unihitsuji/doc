*Help* *Quick Help* : C-h m => *Help*

 Minor modes enabled in this buffer: このバッファで有効なマイナーモード:
Font-Lock Isearch-Fold-Quotes

The major mode is Help mode defined in help-mode.el: メジャーモードは help-mode.el で定義された help-mode です:

Major mode for viewing help text and navigating references in it. ヘルプテキストを表示し、その中の参照をナビゲートするための メジャーモードです。 Also see the help-enable-variable-value-editing variable. また、help-enable-variable-value-editing 変数も 参照してください。

In addition to any hooks its parent mode special-mode might have run, this mode runs the hook help-mode-hook, as the final or penultimate step during initialization. 親モードである special-mode が実行したフックに加えて、 このモードは初期化の最後または最後から2番目のステップとして help-mode-hook フックを実行します。
