document.body.style.backgroundBlendMode の値を取得したいが、 body.onload の段階では、値がない。
window.getComputedStyle(document.body) を .backgroundBlendMode で値を取得できる
DOM 要素の style 属性の設定 (例えば △△.style.display = 〜;) は 属性の値として反映され、その後 (例えば let x = △△.style.display で) 値を取得できる。
△△.style.□□ は HTML のタグに記述されたものしか保持していない。 javascript から設定すれば保持できる。 しかし、style タグ内の指定や外部 CSS による設定は保持していない。
ここから ただの推測: HTML パーサは DOM ツリーの作成に専念し、 style 属性、style タグ、 外部 CSS の指定 <link rel="stylesheet" href="〜"> に 出くわすと CSS パーサに丸投げする。 javascript からの操作は DOM要素の属性 style オブジェクト を通じて行う。 HTML, CSS, javascript の関係について少しだけ詳しくなれた、気がする
